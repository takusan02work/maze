\# 迷路ゲーム (Tilt Maze Game)



スマホを傾けてボールを転がし、ゴールを目指す直感的な迷路ゲームです。



\## 概要

このゲームは、スマートフォンのジャイロセンサー（傾き検知）を利用したアクションパズルゲームです。AIの支援を受けて制作されました。



\## 遊び方

1\. スマートフォンのブラウザでゲーム画面を開きます。

2\. スマホを前後左右に傾けると、ボールがその方向に転がります。

3\. 障害物や壁をうまく避けながら、ゴール地点を目指してください。

※PCブラウザで開いた場合は、マウスやキーボードでは操作できない場合があります。



\## 動作環境・注意点

\- \*\*対応デバイス:\*\* ジャイロセンサー/加速度センサーを搭載したスマートフォン（iPhone/Android）

\- \*\*ブラウザの許可:\*\* iOS（Safari）などの一部の環境では、センサーの利用許可を求めるポップアップが表示されます。「許可」を選択してプレイしてください。

\- \*\*推奨環境:\*\* 横向き・縦向きの固定設定については、端末の設定に従ってください。
